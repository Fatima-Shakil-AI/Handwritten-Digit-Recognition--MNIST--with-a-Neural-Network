# Handwritten Digit Recognition using Neural Networks

This project demonstrates a simple **neural network** implemented in **TensorFlow/Keras** to recognize handwritten digits from the **MNIST dataset**. The model is trained on 60,000 images and tested on 10,000 images, achieving high accuracy in predicting handwritten numbers.

## Features
- Fully-connected neural network with one hidden layer
- Dropout for regularization
- Visualizes training and validation accuracy
- Shows predictions on test images with input vs. predicted labels

## Technologies
- Python
- TensorFlow / Keras
- Matplotlib
- MNIST Dataset

## How to Run
1. Open the notebook in [Google Colab](https://colab.research.google.com/)
2. Run all cells
3. Observe training accuracy and test predictions
